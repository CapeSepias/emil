Syntax Utilities

The emil-common module doesn’t provide utilities for parsing mails or mail addresses from text. This is provided by the emil-javamail module, because the implementation from JavaMail is used.

There is a syntax object that has some common utilities when reading/writing e-mail.

Parsing MimeType from String

import emil.javamail.syntax._
// import emil.javamail.syntax._

import emil._
// import emil._

MimeType.parse("text/html; charset=utf-8")
// res0: Either[String,emil.MimeType] = Right(MimeType(text,html,Map(charset -> utf-8)))

Reading/Writing E-Mail addresses

Parse a string into an e-mail address:

MailAddress.parse("John Doe <jdoe@gmail.com>")
// res1: Either[String,emil.MailAddress] = Right(MailAddress(Some(John Doe),jdoe@gmail.com))

MailAddress.parse("John Doe doe@com")
// res2: Either[String,emil.MailAddress] = Left(Local address contains control or whitespace)

Write the mail address as unicode or ascii-only string:

val ma = MailAddress.parse("Örtlich <über.uns@test.com>").toOption.get
// ma: emil.MailAddress = MailAddress(Some(Örtlich),über.uns@test.com)

val ascii = ma.asAsciiString
// ascii: String = =?UTF-8?Q?=C3=96rtlich?= <über.uns@test.com>

MailAddress.parse(ascii)
// res3: Either[String,emil.MailAddress] = Right(MailAddress(Some(Örtlich),über.uns@test.com))

val unicode = ma.asUnicodeString
// unicode: String = "Örtlich" <über.uns@test.com>

E-Mail from/to String

E-Mail as string:

import cats.effect._
// import cats.effect._

import emil._, emil.builder._
// import emil._
// import emil.builder._

val mail = MailBuilder.build[IO](
  From("test@test.com"),
  To("test@test.com"),
  Subject("Hello"),
  HtmlBody("<p>This is html</p>")
)
// mail: emil.Mail[cats.effect.IO] = Mail(MailHeader(,None,None,Recipients(List(MailAddress(None,test@test.com)),List(),List()),None,Some(MailAddress(None,test@test.com)),None,None,None,Hello,Set()),Headers(List()),Html(IO(<p>This is html</p>)),Attachments(Vector()))

import emil.javamail.syntax._
// import emil.javamail.syntax._

val mailStr = mail.serialize.unsafeRunSync
// mailStr: String =
// Date: Sat, 16 Nov 2019 13:10:45 +0000 (GMT)
// From: test@test.com
// To: test@test.com
// Subject: Hello
// MIME-Version: 1.0
// Content-Type: text/html; charset=us-ascii
// Content-Transfer-Encoding: 7bit
// 
// <p>This is html</p>

Deserialize:

val mail2 = Mail.deserialize[IO](mailStr).unsafeRunSync
// mail2: emil.Mail[cats.effect.IO] = Mail(MailHeader(messageId:null,None,None,Recipients(List(MailAddress(None,test@test.com)),List(),List()),None,Some(MailAddress(None,test@test.com)),None,None,Some(2019-11-16T13:10:45Z),Hello,Set()),Headers(List(Header(MIME-Version,NonEmptyList(1.0)), Header(Content-Type,NonEmptyList(text/html; charset=us-ascii)), Header(Content-Transfer-Encoding,NonEmptyList(7bit)))),Html(IO(<p>This is html</p>)),Attachments(Vector()))
